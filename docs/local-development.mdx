## Prerequisites

- Node.js 20+
- pnpm
- PostgreSQL
- Sui CLI (for contract deployment)

## Backend

```bash
cd backend
pnpm install
cp .env.example .env   # fill in required values
pnpm prisma migrate dev
pnpm dev               # runs on :3000
```

## Frontend

```bash
cd frontend
pnpm install
pnpm dev               # runs on :3001
```

## Required Environment Variables

### Backend (`.env`)

```env
# Core
NODE_ENV=development
PORT=3000
DATABASE_URL=postgresql://...
CORS_ORIGIN=http://localhost:3001

# Auth (user)
JWT_SECRET=
JWT_EXPIRES_IN=7d
GOOGLE_CLIENT_ID=                 # Google OAuth client for zkLogin
ZKLOGIN_PROVER_URL=https://prover-dev.mystenlabs.com/v1

# Sui
SUI_NETWORK=testnet
SUI_RPC_URL=https://fullnode.testnet.sui.io:443
ADMIN_SECRET_KEY=                 # holds AdminCap; signs minting txs (bech32 suiprivkey1...)
GAS_SPONSOR_SECRET_KEY=           # gas sponsor wallet; = ADMIN_SECRET_KEY in Phase 1
GAS_TREASURY_MIN_BALANCE=10000000000   # refuse new txs below this (MIST)
SUI_PACKAGE_ID=                   # deployed arktion package object ID
SUI_ADMIN_CAP_ID=
SUI_ADMIN_REGISTRY_ID=
SUI_INK_TREASURY_CAP_ID=
SUI_EARNING_REGISTRY_ID=
SUI_BADGE_REGISTRY_ID=
SUI_USDC_COIN_TYPE=               # USDC coin type for tips/sends

# Walrus
WALRUS_UPLOAD_RELAY_URL=https://upload-relay.testnet.walrus.space
WALRUS_AGGREGATOR_URL=https://aggregator.walrus-testnet.walrus.space
WALRUS_DEFAULT_EPOCHS=5

# AI (MemWal required — service refuses to start without it)
OPENROUTER_API_KEY=
OPENROUTER_MODEL=meta-llama/llama-3.1-8b-instruct:free
MEMWAL_PRIVATE_KEY=
MEMWAL_ACCOUNT_ID=
MEMWAL_SERVER_URL=https://relayer-staging.memory.walrus.xyz

# Admin auth (separate secret from the user JWT)
ADMIN_JWT_SECRET=                 # min 32 chars; keep distinct from JWT_SECRET
TOTP_ISSUER=Arktion Admin
TOTP_ENCRYPTION_KEY=              # 64 hex chars (32-byte AES-256-GCM key)
```

<Info>
  These names mirror `backend/.env.example` exactly — copy that file and fill the blanks. The app validates them at boot (`config/env.validation.ts`) and refuses to start if a required value is missing or malformed.
</Info>

### Frontend (`.env.local`)

```env
NEXT_PUBLIC_API_URL=http://localhost:3000/api/v1
NEXT_PUBLIC_GOOGLE_CLIENT_ID=
NEXT_PUBLIC_SUI_NETWORK=testnet
```

## Ports

| Service | Port |
|---|---|
| NestJS backend | 3000 |
| Next.js frontend | 3001 |

The API prefix is `/api/v1`. All frontend API calls go through the single axios instance at `lib/api/client.ts`.
